启动

1. docker run [OPTIONS] IMAGE [COMMAND] [ARG...]

2. docker exec [OPTIONS] CONTAINER COMMAND [ARG...]

3. run与exec区别

docker run ：根据镜像创建一个容器并运行一个命令，操作的对象是 镜像

docker exec ：必须在运行的容器中执行命令，操作的对象是 容器
