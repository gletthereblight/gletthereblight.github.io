从 redis 的源码目录中复制 redis.conf 到安装目录，修改 redis.conf 文件:

  • daemonize no : 修改为 yes 表示后台启动
  • bind 127.0.0.1 : 修改为 0.0.0.0，开放给所有 ip
  • protected-mode no : 修改为 yes 表示关闭保护模式，其他客户端也可以连接
  • logfile "" : 修改为相应日志文件，不能是文件夹，默认会将日志写入 /dev/null

3. 停止
