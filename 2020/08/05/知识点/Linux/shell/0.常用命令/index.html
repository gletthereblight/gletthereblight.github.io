Shell 常用命令(持续更新)

1. set

显示系统中已经存在的 shell 变量，以及设置 shell 变量的新变量值，符号”+“和”-“的作用分别是打开和关闭指定的模式

1.1. set -e

Exit immediately if a simple command exits with a non-zero status.

在”set -e“之后出现的代码，一旦出现了返回值非零，整个脚本就会立即退出

1.2. set -x

会在执行每一行 shell 脚本时，把执行的内容输出来，里面涉及的变量也会被替换成实际的值
