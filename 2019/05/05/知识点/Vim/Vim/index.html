Vim 概述

1 模式

  • normal : 任何模式下通过 ESC 进入，此模式下，字母均为功能键
  • insert : normal 下通过 i 进入编辑模式，进行文本操作
  • visual : normal 下通过 v 进入

2 操作

2.1 normal 模式操作

2.1.1 保存

  • :w 保存
  • :q 不保存退出
  • :wq 保存退出

2.1.2 复制粘贴

  • y 复制
  • yy 复制光标所在的整行
  • 2yy 复制以当前行（光标所在行）开始的2行
  • p 粘贴在当前位置之后
  • P 粘贴在当前位置之前
  • d 剪切
  • ^ 选中当前行，光标到行首，类似 HOME 键
  • $ 选中当前行，光标到行尾，类似 END 键

2.1.3 删除插入

  • x 删除当前光标所在一个字符
  • X 删除光标所在前字符
  • dd 删除当前行，并把删除行放到剪切板里
  • 2dd 删除以当前（光标所在行）行开始的2行
  • 2,5d 删除第2到第5行的内容
  • c 删除后进入插入模式
  • a 在光标后插入
  • o 在当前行后插入一个新行
  • O 在当前行前插入一个新行
  • cw 把光标位置到下一个单词之前的内容删掉，并进入插入模式，可以使用c2w删除两个单词

2.1.4 移动

  • 0 数字零，光标移动到行头
  • $ 光标移动到本行行尾
  • w 移动到下一个单词
  • ^ 光标移动到本行第一个不是blank字符的位置
  • g_ 光标移动到本行最后一个不是blank字符的位置

2.1.4 搜索

  • /pattern<enter> 搜索 pattern 的字符串（n到下一个,N到上一个）
