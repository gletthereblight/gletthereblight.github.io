cmd 命令之 dir

1.语法

2.属性

  • /a：显示具有指定属性的文件
名称 意义
/ad 目录详细信息
/ar 只读文件
/ah 隐藏文件
/aa 准备存档文件
/as 系统文件
  • /b：使用空格式(没有标题信息或摘要，即只显示目录的文件名)
  • /c：在文件大小中显示千位数分隔符，是默认值。用 /-c 来禁用分隔符显示
  • /d：用宽列表格式列出，但文件是按栏分类列出的（纵向排序）
  • /l：目录名用小写字母显示
  • /n：新的长列表格式，其中文件名在最右边，是默认值
  • /o：用分类顺序列出文件
名称 意义
/on 按名称(字母顺序)
/os 按大小(从小到大)
/oe 按扩展名(字母顺序)
/od 按日期/时间(从先到后)
/as 组目录优先
  • /p：在每个信息屏后暂停
  • /q：显示文件所有者
  • /r：显示文件的备用数据流
  • /s：显示指定目录和所有子目录中的文件
  • /t：控制显示或用来分类的时间字符域
名称 意义
/tc 创建时间
/ta 上次访问时间
/tw 上次写入时间
  • /w：用宽列表格式（横向排序）
  • /x：显示 NTFS 和 FAT 卷上文件生成的短名称。格式是 /n 的格式，短名称插在长名称前面。如果没有短名称，在其位置则显示空白
  • /4：用四位数字显示年。
