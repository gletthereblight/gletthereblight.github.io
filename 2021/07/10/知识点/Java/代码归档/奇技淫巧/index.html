Java 奇技淫巧

数组 Array

在 java.lang.reflect.Array 中提供了几个实用方法，用来动态创建和使用 java 数组，其实现基本都是 Native 方式

1. newInstance

Array.newInstance(Class<?> componentType, int length)，使用提供的类型和长度来创建一个数组

2. set

Array.set(Object array, int index, Object value)，向指定数组的下标位置添加一个元素。还有许多指定类型的添加方法，如 setInt，setDouble，setChar 等

3. get

Array.get(Object array, int index)，获取指定位置的元素。还有许多指定类型的添加方法，如 getInt，getDouble，getChar 等
