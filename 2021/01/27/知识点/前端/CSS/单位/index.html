CSS 中常用单位

1. 单位

1. px

绝对单位，意思是一个光点，和当前操作系统视窗分辨率有关。

比如 1920*1080 的 16:9 的分辨率，代表荧幕横向有1920个光点、纵向有1080个光点。1px代表一个光点。

2. em

和父级元素的字体大小有关，即父级元素的字体大小的 n 倍的意思，如果父级元素没有设置，一直找到 html 顶级元素为止

3. rem

即 root em，只会根据根标签 <html> 计算字体大小的倍数

4. vh/vw

即 viewport width 和 viewport height，和浏览器当前尺寸有关，即浏览器当前长宽的百分比，范围是 1~100

5. vmin/vmax

即 viewport min 和 viewport max，前者会选用荧幕较短一边的总边长作为宽度，后者则是使用较长边，和设备当前分辨率有关，v的意思同vh/vw，即百分比，和父级元素的size有关
