Mysql 零碎

1. InnoDB 索引

1. 索引分类

1. 聚簇索引(clustered index)

聚集索引的叶子节点用来存储行记录，因此有且只有一个聚集索引：

  • 如果定义了主键，则其为聚簇索引
  • 如果未定义主键，则第一个 not NULL unique 列是聚集索引
  • 否则，InnoDB 会创建一个隐藏列作为聚集索引

2. 二级索引(secondary index)

二级索引的叶子节点只包含索引列和主键

2. Mysql 各个版本区别

  • Oracle MySQL Cloud Service: 建立在企业版上，使用oracle云提供核心的云服务
  • MySQL Enterprise Edition: 企业版本，需付费，可以试用30天
  • MySQL Cluster CGE: 集群版，开源免费。可将几个MySQL Server封装成一个Server
  • MySQL Community Edition: 社区版本，开源免费，但不提供官方技术支持
