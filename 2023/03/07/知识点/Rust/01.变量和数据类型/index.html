Rust 基础-变量和数据类型(第一章)

1. 变量和常量

1. wiki

  • 变量 : 声明时用 let 关键字，默认不可变，但可再增加 mut 声明成可变
  • 常量 : 声明时用 const 关键字，且必须标注类型，可以声明在任何作用域里。只能绑定常量表达式。规范是全部大写，且用下划线连接多个单词。

2. 特点

  • 变量隐藏 : 可以使用相同名字声明新变量，此时会将之前的声明覆盖掉。

2. 数据类型

1. wiki

作为静态语言，编译时需要知道其类型，当可能得类型比较多时(如字符串转数字)，不指定类型就会编译报错。

2. 类型

1. 标量类型

代表单值，有四种

1. 整数类型

有无符号分为 i 和 u，长度为 8,16,32,64,128bit，i32为默认类型。特殊的有 isize 和 usize，根据系统架构决定(如32位计算机长度为32bit)。

对于整数类型溢出，当使用调试模式时，会抛出 panic，发布模式下则会直接环绕(如 u8 时，256变0，257变1)

2. 浮点类型

分为2种，f32 单精度32位，f64 双精度64位，后者是默认类型。

3. 布尔类型

符号为 bool，占1个字节，分为2种，true 和 false。

4. 字符类型

符号为 char，占4个字节，字面量使用单引号。

2. 复合类型

1. Tuple 元组

可以存放多种数据类型，且长度是固定的，一旦声明就无法改变

2. 数组

只能存放单一数据类型，且长度是固定的，存在栈内存中。下标溢出时，编译通过，但运行时报错
